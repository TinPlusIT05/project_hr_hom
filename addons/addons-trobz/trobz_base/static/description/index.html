Features available in all projects at Trobz

Module Kanban View Optimization

In the kanban view, add a button 'Upgrade' for 'Installed' modules to easily Upgrade module in Kanban View.

Available functions which are often used in projects

  • update_company_logo: Update Logo For Company
  • update_config: Update Settings For Sale Settings, Purchase Settings, ...
  • unlink_object_by_xml_id: Remove Object by XML ID
  • delete_default_products: Delete Several Product With Following Name
  • run_post_object_one_time: Run functions Only One Time To Create/Update Data
  • make_default_manager_groups: create Demo manager profile

Update Separator Format for Language

  • Default is en_US.UTF-8
