Odoo Multi Process Logger

  • Normally, default logger of Odoo can't handle log very well with multi-process (workers). So this module helps to replace native Odoo log handler by another that can log well in multi-process environment
  • To use this module, should do following steps:
      1. Install library safe_logger (define it in requirements in your project or install it directly)
      2. Load it when starting Odoo server by --load worker-logger
